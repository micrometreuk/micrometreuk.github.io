#### Install Node.js (LTS) on  Ubuntu debian based linux distribution


```bash
curl -sL https://deb.nodesource.com/setup_12.x | sudo -E bash -
sudo apt-get update
sudo apt-get install -y nodejs
```

For the latest installation guide on Node.js website
https://nodejs.org
